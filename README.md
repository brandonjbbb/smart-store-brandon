🛍️ Smart Store - Customer Revenue Analysis

🚀 Project to analyze total revenue per customer and provide insights for business decisions.

📌 Project Overview
This project calculates the total revenue generated by each customer to help businesses identify their most valuable customers.
This data can be used for loyalty programs, marketing strategies, and sales improvements.

📂 Project Structure

smart-store-brandon/
│
├── data/                     # Stores datasets
│   ├── raw/                  # Raw data before processing
│   ├── customer_total_revenue.csv  # Generated revenue report
│
├── scripts/                  # Python scripts
│   ├── main.py               # Main analysis script
│
├── utils/                    # Helper scripts
│   ├── logger.py             # Logging functions (for future use)
│
├── .gitignore                # Files to ignore in version control

⚙️ Setup Instructions

Follow these steps to set up and run the project.

1️⃣ Clone the Repository
git clone https://github.com/brandonjbbb/smart-store-brandon.git
cd smart-store-brandon
2️⃣ Create a Virtual Environment
python3 -m venv venv
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate     # Windows
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Run the Script
python scripts/main.py
✅ This will calculate total revenue per customer and save it in data/customer_total_revenue.csv.
